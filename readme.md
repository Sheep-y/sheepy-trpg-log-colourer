# Sheepy's Chat Log Colourer #

Apply speaker-based BBCode colouring to textual chat logs.

Main features:
  * Single HTML file, can be used online or offline.
  * Auto detection of date format, names, main speakers, and dice bots.　
  * Auto remember last used colouring.
